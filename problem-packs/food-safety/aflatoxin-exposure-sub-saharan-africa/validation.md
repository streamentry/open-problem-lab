# Validation

## Validation Layers

| Layer           | Gate                                                                 |
| --------------- | -------------------------------------------------------------------- |
| Structure       | `pnpm validate` passes schemas and problem-pack completeness checks  |
| Evidence        | Every claim has a dated evidence record with method and limitations  |
| Reproducibility | Contamination estimates include rerunnable data compilation steps    |
| Review          | Required reviewer roles approve or request changes                   |
| Replication     | Safety-sensitive or quantitative claims are independently reproduced |

## Baseline Requirements

District-level contamination estimates must document:

- District identifier.
- Grain type.
- Test method and detection limit.
- Sampling frame.
- Collection date and season.
- Number of samples.
- Missingness handling.
- Administrative-boundary changes.

## Risk Map Requirements

No risk map may be called actionable until it includes:

- Input data sources with sampling frame documentation.
- Contamination threshold source (Codex, national standard, or stated).
- Geographic extrapolation method and its limits.
- Uncertainty range or confidence statement.
- Known non-use cases.
- Replication status.

## Intervention Review Requirements

No intervention recommendation may be merged without:

- Named decision-maker.
- Named allocation decision it would change.
- Efficacy evidence with study type and context.
- Cost estimate or range.
- At least one failure mode.
- Non-use cases.
