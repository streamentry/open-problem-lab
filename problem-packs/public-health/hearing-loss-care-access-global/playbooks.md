# Playbooks

## Analyst Playbook

1. Start with `problem.json`, `evidence.json`, and `tasks.json`.
2. Select one scoped task.
3. Add evidence before adding claims.
4. For workforce density, verify the audiologist definition matches across sources. Different countries use different titles and training pathways.
5. For coverage gap estimates, document the estimation method explicitly. A proxy estimate is acceptable if labeled as such.
6. Keep quantitative outputs reproducible.

## Reviewer Playbook

Ask five questions:

1. What claim is being made?
2. What source proves it?
3. What method produced it?
4. What would make it false?
5. Who could misuse it?

## Field-Reality Playbook

Do not ask whether a coverage gap number is impressive. Ask whether it would change a named health ministry's resource allocation decision at a useful time, without creating worse incentives. A coverage gap that only proves "the problem is big" without pointing to a specific allocation lever is not field-ready.
