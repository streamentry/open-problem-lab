# Validation

## Validation Layers

| Layer           | Gate                                                                 |
| --------------- | -------------------------------------------------------------------- |
| Structure       | `pnpm validate` passes schemas and problem-pack completeness checks  |
| Evidence        | Every claim has a dated evidence record with method and limitations  |
| Reproducibility | Exposure estimates include rerunnable WBGT calculation steps         |
| Review          | Required reviewer roles approve or request changes                   |
| Replication     | Safety-sensitive or quantitative claims are independently reproduced |

## Baseline Requirements

Sub-national WBGT exposure estimates must document:

- Grid or station identifier.
- Calculation method (which WBGT formula, indoor or outdoor).
- Time period and temporal resolution.
- Outdoor worker population denominator by industry where available.
- Employment status (formal, informal) coverage.
- Missingness handling.
- Administrative-boundary changes.

## Exposure-Outcome Requirements

No exposure-outcome relationship may be called actionable until it includes:

- Input exposure data sources with metric and grain.
- Health outcome data source and surveillance method.
- Effect size with confidence interval.
- LMIC transferability assessment.
- Known non-use cases.
- Replication status.

## Intervention Review Requirements

No intervention recommendation may be merged without:

- Named decision-maker.
- Named allocation decision it would change.
- Efficacy evidence with study type and context.
- Cost estimate or range.
- At least one failure mode.
- Non-use cases.
